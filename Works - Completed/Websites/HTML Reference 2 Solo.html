Headings - H2 Example

Headings are defined from H1 to H6. H1 is the largest, example on the title of the page, h6 is the smallest.

Paragraphs

Paragraphs are done with the Tag P. What is not carried over is the formating. Like spaces and enters. This is done using the
BR tag to give your text a break line. And the

HR tag where it creates a Horizontial line on your document.

There is also the PRE tag. 
                        This creates a Preformated text, 
                        maintaining all spaces, lines
                        and format.

Text formating

Usually this is done using CSS - Cascading Style Sheets, but for demonstration reasons i will list them here.

  1. Bold text using the B tags.
  2. Emphasized text using the EM tag.
  3. Alternate voice or mood text with the i tag,
  4. Small text using the Small tag.
  5. Strong text using the STRONG tag
  6. subscripted text with SUB tag.
  7. Superscripted text with SUP tag.
  8. Inserted text using the INS tag.
  9. Deleted text with the DEL tag.
  10. Marked text using the MARK tag.

Some text extras, inline level elements

  • ABBR for abbreviations.
  • Address for contact information for the author/owner of a document.
  • BDO for text direction.
  • BlockQuote for Quotations.
  • CITE for Citations
  • And Q for inline Quotations

HTML Tag Attributes

All HTML elements can have attributes. Here is some of them.

  1. The href attribute of a defines the URL of the page the link goes to.
  2. Also the target of a specifies where to open the linked document.
  3. And the mailto: scheme inside the href attribute to create a link that opens the user's email program.
  4. The src attribute of img specifies the path for the image to be displayed.
  5. The width and height attributes of img provide size information for images.
  6. The alt attribute of img provides an alternate text for an image
  7. The lang attribute of the HTML tag declares the language of the Web page.
  8. The title attribute defines some extra information about an element
  9. The style attribute is used to add styles to an element, such as color, font, size. More on CSS Reference.

And some last ones. (for now )

  • The Hgroup tag defines a group of tags
  • Interactive: Content specifically intended for user interaction.
    Button, Details, Embed, iframe, input, label, object, select and text area.
  • Embedded: Content that imports other resources into the document.
    Audio, video, canvas, iframe, img, math, object, svg
  • Metadata: Content that sets up the presentation or behavior of the rest of the content. These elements are found in the head of the document.
    Base, link, meta, noscript, script, style.
